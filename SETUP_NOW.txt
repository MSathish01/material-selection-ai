╔══════════════════════════════════════════════════════════════════════════════╗
║ STEP 1: Get MongoDB Connection String (15 minutes)                          ║
╚══════════════════════════════════════════════════════════════════════════════╝

Option A: MongoDB Atlas (Cloud - RECOMMENDED)
────────────────────────────────────────────────

1. Go to: https://www.mongodb.com/cloud/atlas/register
2. Sign up (it's FREE)
3. Create a FREE cluster (M0)
4. Create database user:
   - Username: materialuser
   - Password: (create a strong password and SAVE IT!)
5. Whitelist IP: Click "Allow Access from Anywhere"
6. Get connection string:
   - Click "Connect" → "Connect your application"
   - Copy the string (looks like):
     mongodb+srv://materialuser:<password>@cluster0.xxxxx.mongodb.net/...
   - Replace <password> with your actual password

Option B: Use Demo Connection (For Quick Testing)
──────────────────────────────────────────────────

If you want to test quickly, you can use a demo connection temporarily.
(Not recommended for production)

╔══════════════════════════════════════════════════════════════════════════════╗
║ STEP 2: Get AI API Key                                                      ║
╚══════════════════════════════════════════════════════════════════════════════╝

🎉 NEW: You can now use Google Gemini API for FREE!

Option A: Google Gemini API (FREE - RECOMMENDED)
─────────────────────────────────────────────────

1. Go to: https://makersuite.google.com/app/apikey
2. Sign in with your Google account (Gmail)
3. Click "Create API Key"
4. Copy the key (starts with AIza)
5. SAVE IT SECURELY!

✅ FREE - No credit card required!
✅ 60 requests/minute
✅ 1,500 requests/day
✅ Perfect for testing and development

See GET_GEMINI_API_KEY.md for detailed instructions.

Option B: OpenAI API (Paid)
────────────────────────────

1. Go to: https://platform.openai.com/signup
2. Sign up or log in
3. Add payment method (minimum $5)
4. Go to: https://platform.openai.com/api-keys
5. Click "Create new secret key"
6. Copy the key (starts with sk-)
7. SAVE IT SECURELY!

╔══════════════════════════════════════════════════════════════════════════════╗
║ STEP 3: Configure the .env File                                             ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. Open the .env file in Notepad:
   
   notepad .env

2. Update these lines:

   # If using Gemini (FREE):
   GEMINI_API_KEY=AIza...your-actual-key-here
   
   # OR if using OpenAI (Paid):
   # OPENAI_API_KEY=sk-your-actual-key-here
   
   # MongoDB connection:
   MONGODB_URI=mongodb+srv://materialuser:yourpassword@cluster0.xxxxx.mongodb.net/material-selection

3. Save and close (Ctrl+S)
