🌐 AFTER LOCAL WORKS, DEPLOY TO FIREBASE:

STEP 1: Create Firebase Project
  → Go to: https://console.firebase.google.com/
  → Click "Add project"
  → Name it: material-selection-ai
  → Click "Create project"
  → COPY THE PROJECT ID (e.g., material-selection-ai-abc123)

STEP 2: Update .firebaserc
  → Open .firebaserc file
  → Replace "your-project-id" with YOUR ACTUAL PROJECT ID
  → Save the file

STEP 3: Initialize Firebase
  → Run: firebase init
  → Select: Hosting
  → Choose: Use existing project
  → Select: Your project from list
  → Public directory: frontend/build
  → Single-page app: Yes
  → Overwrite: No

STEP 4: Deploy
  → Run: cd frontend && npm run build && cd ..
  → Run: firebase deploy --only hosting
  → Your app is live!
